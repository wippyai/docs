# 贡献

Wippy 由 [Spiral Scout](https://spiralscout.com) 开发。我们在内部系统和客户项目中将其用于生产环境。公开发布版本处于 alpha 阶段。

## 稳定性

- **Lua API**：稳定
- **外部行为**：稳定
- **Registry entry 定义**：稳定，有少量例外。如需更改会提供迁移路径。
- **内部运行时 API**：可能更改
- **底层机制**：可能更改

## Pull Request

我们接受关于 Lua 模块改进、API 稳定化和文档的 pull request。

我们不接受关于高级架构变更或核心设计决策的 pull request。在此阶段我们需要保持对主要方向的控制。

欢迎随时提出 issue 和反馈。

## 其他项目

Spiral Scout 维护多个开源项目，包括 [RoadRunner](https://github.com/roadrunner-server/roadrunner)、[Spiral Framework](https://github.com/spiral/framework) 和 [Cycle ORM](https://github.com/cycle/orm)。

## 安全

安全漏洞请直接发送邮件至 security@wippy.dev。

## 商业支持

如需商业支持或提前访问，请联系 wippy@spiralscout.com。
