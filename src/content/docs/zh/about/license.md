# 许可证

Wippy 基于 Mozilla Public License 2.0 发布。

## Mozilla Public License 2.0

```
Copyright 2025 Spiral Scout

This Source Code Form is subject to the terms of the Mozilla Public
License, v. 2.0. If a copy of the MPL was not distributed with this
file, You can obtain one at https://mozilla.org/MPL/2.0/.
```

## 这意味着什么

你可以在商业项目中使用 Wippy，修改源代码，并分发你的修改。修改的文件必须保持 MPL-2.0 许可，但你可以在更大的作品中将 Wippy 与专有代码结合使用。

分发修改的 MPL 许可文件时，必须提供其源代码。

## 依赖

Wippy 使用各种开源依赖，每个都有自己的许可证。所有依赖都与商业使用兼容。
